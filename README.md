## To run the api local

- Navigate to `/restapi`
- Run `npm i`
- Run `sudo mongod -dbpath ./data/db` to start the local database
- Run `nodemon index.js` to start the node local server

- Navigate to `/webapp`
- Run `npm i`
- Run `npm start`
- Open localhost:8081 if it does not opened automatic
